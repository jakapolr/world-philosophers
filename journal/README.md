# Growth Journal

This folder holds the **growth-mirror** journal — a record of meaningful mentorship
sessions over time, so you can *see* how you change.

- The plugin's `growth-mirror` skill writes entries here (one file per session:
  `<YYYY-MM-DD>-<slug>.md`, following `templates/growth-entry-template.md`).
- Run `/reflect` to have the council mirror your journey back to you.
- Run `/reflect log` to record the session you just had.

> **Privacy.** These entries are personal. The repo's `.gitignore` excludes everything
> here except this README and any file named `*.example.md`. If you fork this project,
> your reflections stay on your machine — they are never committed to a public repo.
>
> `seed-the-heart.example.md` is kept on purpose as the project's first, founding entry.
